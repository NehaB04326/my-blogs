Master the IBPS PO 2025 Application Process in 15 Minutes

Nearly 25% of IBPS PO applications get rejected every year, not because candidates lack qualifications, but because of errors in the online application form. A wrong photograph dimension, missing handwritten declaration, or incorrect fee payment can cost you an entire year.

The IBPS PO 2025 application process involves 8 crucial steps: registration, document upload, form filling, preview, handwritten declaration submission, fee payment, final submission, and taking printout. Each step has specific requirements that must be followed precisely.

This guide walks you through the complete application process for IBPS PO 2025 with screenshots, document specifications, payment methods, and common errors to avoid. Follow these instructions carefully to ensure your application is accepted without rejection.

Before You Start: Documents You Need

Preparing all documents before starting the application saves time and prevents incomplete submissions. Gather these items first:

Digital Documents Required

Scanned Photograph:

  • File format: JPG/JPEG only
  • File size: 4 KB to 12 KB
  • Dimensions: 200 x 230 pixels (width x height)
  • Image quality: Clear, recent passport-size photo
  • Background: Light colored, preferably white

Scanned Signature:

  • File format: JPG/JPEG only
  • File size: 1 KB to 12 KB
  • Dimensions: 140 x 60 pixels (width x height)
  • Requirements: Sign on white paper with black ink pen

Left Thumb Impression:

  • File format: JPG/JPEG
  • File size: 10 KB to 20 KB
  • Dimensions: 240 x 240 pixels (exact square)
  • Requirements: Clear impression on white paper, scan in grayscale

Handwritten Declaration:

  • File format: JPG/JPEG
  • File size: 50 KB to 100 KB
  • Dimensions: 800 x 400 pixels
  • Requirements: Write in English on white paper in your own handwriting (not capital letters)

Important: Many candidates fail at document upload because file sizes don't match specifications. Use image editing tools or online resizers to adjust file sizes before uploading.

Other Information Needed

  • Valid email ID (active throughout recruitment process)
  • Active mobile number (for OTP verification)
  • Educational certificates (for details like university name, year, percentage)
  • Category certificate if claiming reservation
  • PwD certificate if applicable

Step-by-Step IBPS PO 2025 Application Process

Step 1: Visit the Official IBPS Website

Go to www.ibps.in and locate the CRP PO/MT section on the homepage. Click on "Common Recruitment Process for Probationary Officers/Management Trainees (CRP-PO/MT-XV)".

The official website is the ONLY platform for applications. Avoid third-party websites claiming to assist with registration—they're either scams or will charge unnecessary fees.

Step 2: Click on New Registration

On the application page, find the button "CLICK HERE FOR NEW REGISTRATION" located at the top right corner. Click this to begin your registration process.

If you previously registered for any IBPS exam, DO NOT use old credentials. Each recruitment requires fresh registration with a new registration number.

Step 3: Complete Basic Registration

Enter the following details in the registration form:

  • Full name (exactly as per Class 10 certificate)
  • Date of birth (DD-MM-YYYY format)
  • Email ID
  • Mobile number
  • Security code/CAPTCHA

Click "Save & Next" after filling all fields. A provisional registration number and password will be sent to your registered email ID and mobile number via SMS. Save these credentials immediately—you'll need them to log in and complete your application.

Step 4: Log In and Upload Documents

Return to the application page and click "Login for Already Registered Candidates". Enter your registration number and password received earlier.

Now upload all scanned documents in the exact specifications mentioned above. The system will reject files that don't meet size or dimension requirements.

Pro Tip from Experience: Simran from Delhi shares: "I wasted 2 hours because my photo was 15 KB. I had to compress it multiple times to bring it under 12 KB. Prepare documents according to exact specifications beforehand."

Step 5: Fill the Detailed Application Form

The application form has multiple sections. Fill each carefully:

Personal Details:

  • Father's/Mother's name
  • Gender, marital status
  • Nationality, religion, category
  • Correspondence and permanent address
  • State of domicile

Educational Qualifications:

  • Degree name (BA/B.Com/B.Sc/B.Tech, etc.)
  • University name
  • Year of passing
  • Percentage/CGPA (convert CGPA to percentage using university formula)

Work Experience (Optional):

  • Mention if you have any work experience
  • This field is not mandatory

Category and Reservation Details:

  • Select your category (General/OBC/SC/ST/EWS/PwD)
  • Upload relevant certificates if claiming reservation

Bank Preferences:

  • Select up to 3 bank preferences in order of priority
  • Choose carefully—final allotment depends on these preferences

Pre-Examination Training (PET):

  • SC/ST/Minority candidates can opt for free pre-exam training
  • Select "Yes" if interested, "No" if not required

Step 6: Preview Your Application

After completing all sections, click "Preview" to review your entire application. Check every detail carefully:

  • Name spelling matches Class 10 certificate exactly
  • Date of birth is correct
  • Email and mobile number are active
  • Educational percentage calculations are accurate
  • Category selection is correct
  • Bank preferences are in desired order

If any errors are found, click "Edit" to make corrections. You can edit unlimited times before final submission.

Step 7: Upload Handwritten Declaration

Write the following declaration in your own handwriting in English on white paper:

"I, __________ (your name), hereby declare that all the information submitted by me in the application form is correct, true and valid. I will present the supporting documents as and when required."

Critical Instructions:

  • Write in cursive/running handwriting, NOT capital letters
  • Use black ink pen
  • Write on plain white paper
  • Scan and upload as per specifications (800 x 400 pixels, 50-100 KB)

This declaration authenticates your application. Missing or improperly formatted declarations lead to immediate rejection.

Step 8: Pay Application Fee

IBPS PO application fee structure:

Category

Application Fee

General / OBC / EWS

₹850

SC / ST / PwD

₹175

The fee includes examination charges plus intimation charges. Once paid, this fee is non-refundable under any circumstances.

Payment Methods Available:

  • Debit Card (RuPay/Visa/MasterCard/Maestro)
  • Credit Card
  • Net Banking
  • UPI
  • Mobile Wallets (Paytm, PhonePe, etc.)
  • IMPS

After selecting your payment method, you'll be redirected to the payment gateway. Complete the transaction and wait for the confirmation message. The system generates an e-receipt after successful payment.

Payment Tip: Use net banking or UPI for instant confirmation. Card payments sometimes face gateway issues during peak hours.

Step 9: Final Submission

After successful payment, click "Complete Registration" or "Submit" button. Once submitted, you CANNOT make any changes except during the correction window.

The system displays a confirmation message: "Your application has been successfully submitted." Take a screenshot of this confirmation page.

Step 10: Download and Print

Download your completed application form and payment receipt. Take at least 2-3 printouts and keep them safely for future reference. You'll need these during:

  • Document verification
  • Interview stage
  • Any queries regarding your application

The printout serves as proof that you successfully applied within the deadline.

IBPS PO Application Correction Window

IBPS typically opens a 2-day correction window after the application deadline closes. For 2025, the correction window was July 31 to August 1.

What Can Be Corrected?

During the correction window, you can edit:

  • Address details
  • Educational qualification details
  • Category (with documentary proof)
  • Bank preferences
  • Examination center preferences

What CANNOT Be Corrected?

These fields are permanently locked:

  • Name
  • Date of birth
  • Email ID
  • Mobile number

Correction Fee: ₹200 must be paid to access the correction facility. You can make corrections only ONCE, so review all changes carefully before resubmitting.

Ravi from Hyderabad warns: "I forgot to select my preferred bank during application. During the correction window, I paid ₹200 to change it. Double-check everything during first submission itself."

Common Application Mistakes to Avoid

Based on analysis of rejected applications, these are the top errors:

Document Upload Failures:

  • Photo/signature exceeding file size limits
  • Wrong image dimensions
  • Using PNG/PDF instead of JPG format
  • Unclear or blurry scanned images

Form Filling Errors:

  • Name spelling doesn't match Class 10 certificate
  • Incorrect date of birth
  • Wrong category selection
  • Percentage calculation errors in CGPA conversion
  • Invalid email ID or mobile number

Payment Issues:

  • Not waiting for payment confirmation
  • Closing browser during payment gateway processing
  • Not saving payment receipt
  • Double payment due to page refresh

Declaration Problems:

  • Writing in capital letters
  • Someone else writing on your behalf
  • Not following exact text provided by IBPS
  • Wrong file size or dimensions

What Happens After Successful Application?

Once your application is accepted, IBPS will:

  1. Send Confirmation Email: You'll receive an email acknowledging your application within 24-48 hours
  2. Issue Admit Card: Prelims admit card is released 2-3 weeks before the exam date
  3. Send SMS Alerts: Important updates are sent via SMS to your registered mobile number

Keep checking your email and the official IBPS website regularly for updates. Admit cards are NOT sent by post—you must download them from the website.

Technical Issues During Application

If you face technical problems:

Browser Issues:

  • Use updated versions of Chrome, Firefox, or Edge
  • Clear browser cache and cookies before starting
  • Disable pop-up blockers
  • Ensure stable internet connection

Payment Gateway Errors:

  • Wait 24 hours for payment status to update
  • Check bank statement for debit confirmation
  • If amount deducted but registration incomplete, contact IBPS helpdesk with transaction details
  • Don't make duplicate payments immediately

Document Upload Problems:

  • Check file format is JPG/JPEG, not PNG or PDF
  • Use online image resizers to adjust file size
  • Ensure images are clear and not corrupted
  • Try uploading during non-peak hours

Frequently Asked Questions

1. Can I apply for IBPS PO offline?

No, IBPS PO accepts only online applications through the official website www.ibps.in. There is no offline application process available. Candidates must have a valid email ID, mobile number, and internet access to complete the online registration. Any offline applications or applications sent via post will not be accepted under any circumstances.

2. What if I make a mistake in my IBPS PO application form?

If you haven't submitted the form yet, you can edit it unlimited times before final submission. After submission, you must wait for the correction window (usually 2 days after application deadline) to make changes. A correction fee of ₹200 applies, and you can correct only once. However, name, date of birth, email ID, and mobile number cannot be changed even during the correction window.

3. How long does the IBPS PO application process take?

If all documents are prepared according to specifications, the complete application process takes 20-30 minutes. However, if you need to resize images, convert CGPA to percentage, or arrange documents, it may take 1-2 hours. Complete the process in one sitting without closing the browser to avoid losing entered data.

4. Is the application fee refundable if I don't appear for the exam?

No, the IBPS PO application fee is non-refundable under any circumstances. Whether you appear for the exam or not, qualify or not, or your application is rejected—the fee will not be refunded. It also cannot be adjusted against future IBPS examinations. Ensure you meet all eligibility criteria before paying the fee.

5. Can I apply for IBPS PO from mobile phone?

While it's technically possible to apply from a mobile phone, it's not recommended. Document upload, form filling, and payment processes work better on desktop/laptop computers. Mobile browsers may have compatibility issues with the IBPS portal. If you must use mobile, ensure you're on a stable internet connection and use Chrome or Firefox mobile browsers.
